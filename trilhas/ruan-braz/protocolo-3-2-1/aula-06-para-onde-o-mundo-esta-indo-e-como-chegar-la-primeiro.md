# Para onde o mundo está indo e como chegar lá primeiro

**Tempo estimado de leitura:** 3 minutos

## Objetivos de aprendizado

Ao final desta aula, você será capaz de:

- Identificar como as empresas de IA monetizam os dados dos usuários e por que as versões gratuitas existem
- Distinguir entre o que as versões gratuitas e pagas das IAs oferecem
- Aplicar o uso de documentação oficial como fonte de aprendizado para qualquer ferramenta de IA
- Reconhecer a documentação como um recurso acessível, não exclusivo para desenvolvedores

## O modelo de negócio das IAs: dados, gratuidade e limitações

A principal moeda do mundo hoje são os dados. As empresas que mais têm dados têm mais chances de construir modelos de IA mais poderosos, porque usam essa base para treinar os modelos e conseguir resultados cada vez melhores. O desafio é que essas empresas já rasparam toda a internet disponível.

Para continuar treinando, muitas recorrem a dados sintéticos: geram dados com inteligência artificial para alimentar a própria IA. Mas a principal estratégia é outra: precisam de usuários que utilizem as plataformas e gerem mais dados. Em outras palavras, as principais ferramentas de IA do mundo querem que você as use para que continuem treinando com o que você produz.

As IAs mais poderosas do mundo pertencem às empresas com mais dados. Entre as citadas: Meta, identificada como a empresa que mais tem dados no mundo hoje, Google, ByteDance, OpenAI e Claude. Essas empresas investem fortemente em dados e querem que o usuário assine para sustentar os custos com servidores, além de usar as plataformas ao máximo.

É exatamente por isso que existem versões gratuitas: ao usar uma IA gratuitamente, o usuário está pagando com acesso e entregando dados. Ainda assim, o custo de manter os servidores é muito alto, então o uso nas versões gratuitas é limitado: os modelos disponíveis são mais fracos e entregam resultados inferiores. Quem assina tem acesso a modelos melhores, e a diferença de qualidade é significativa.

## Documentação oficial: o melhor manual para aprender qualquer ferramenta

Documentação não é exclusiva de desenvolvedores. Ela é o melhor caminho para interpretar qualquer ferramenta de IA: é um manual passo a passo escrito pelas próprias pessoas que criaram a ferramenta.

Para acessar a documentação do ChatGPT, basta digitar "ChatGPT Documentation" no Google, entrar no site API Platform da OpenAI e clicar em "Comece a Criar". A plataforma apresenta uma documentação completa com todos os modelos disponíveis, suas vantagens e custos para uso via API. Quem assina o ChatGPT não paga esses custos de API. Também estão disponíveis manuais para geração de texto, imagem e áudio. Mesmo que a página inicial pareça avançada por conter código, ao rolar para baixo aparecem conteúdos mais acessíveis: construção de agentes, geração de imagens com explicações de como funcionam, e muito mais.

O mesmo funciona para qualquer modelo. Ao buscar "Midjourney Documentation", o usuário encontra a documentação completa da ferramenta, incluindo guia para começar, engenharia de prompts e como escrever prompts básicos. Se a documentação estiver em inglês, basta usar o recurso de tradução do navegador: o conteúdo funciona perfeitamente traduzido, e não existe nenhum impedimento para aprender qualquer IA com essas fontes.

## Coloque em prática

Acesse a documentação oficial de uma ferramenta de IA que você usa ou quer usar.

Navegue sem pressa: se parecer avançado, role para baixo até encontrar o conteúdo que faz sentido para o seu nível agora.

Se estiver em inglês, ative a tradução automática do navegador e siga a leitura.
