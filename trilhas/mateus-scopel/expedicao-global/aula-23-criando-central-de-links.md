# Criando Central de Links

**Tempo estimado de leitura:** 4 minutos

## Objetivos de aprendizado

Ao final desta aula, você será capaz de:

- Estruturar uma central de links para acelerar o envio de propostas.
- Identificar categorias úteis para organizar projetos de portfólio.
- Aplicar tags e descrições curtas para localizar exemplos rapidamente.
- Reconhecer como uma boa organização reduz o tempo entre encontrar uma vaga e enviar a proposta.

## Visão geral

Nesta aula, Mateus apresenta uma dica prática para ganhar velocidade na hora de enviar propostas: criar uma central de links.

A ideia é reunir projetos de portfólio, links, descrições e tags em um documento único. Assim, quando surgir uma vaga compatível, você encontra rapidamente os exemplos certos para incluir na candidatura.

Essa organização é especialmente útil para quem está testando diferentes serviços, como identidade visual, design gráfico, pixel art, apresentação, editorial ou cartão de visita.

## Por que criar uma central de links

Na Upwork, velocidade importa. Quanto antes você envia uma boa proposta para uma vaga adequada, maior tende a ser sua chance de ser visto.

Sem organização, você perde tempo procurando links em vários lugares, abrindo pastas, caçando projetos antigos ou tentando lembrar qual exemplo combina melhor com cada cliente.

A central de links resolve esse problema. Ela deixa seus melhores materiais prontos para uso, separados por tipo de serviço e fáceis de encontrar.

Mateus reforça que essa não é uma etapa obrigatória, mas pode aumentar bastante a agilidade do processo.

## Como organizar as categorias

A central pode ser criada em um documento simples, como Google Docs ou outro editor de texto.

Mateus mostra uma estrutura dividida por categorias, como identidade visual, design gráfico e pixel art.

Dentro de cada categoria, você pode criar subcategorias mais específicas. Em identidade visual, por exemplo, podem existir marcas, eventos, campanhas ou identidades para ocasiões específicas.

Em design gráfico, pode haver cartão de visita, editorial, book design e outros formatos.

Essa divisão ajuda a localizar rapidamente o projeto certo para cada vaga.

## O que incluir em cada projeto

Para cada item, Mateus sugere incluir o nome do projeto, o link do portfólio e uma descrição curta.

O link pode ser da própria Upwork, que é a opção recomendada. Se não for possível, também pode ser um link do Drive, Behance ou outro repositório visual.

A descrição é opcional, mas ajuda a criar uma base para o texto que será usado na proposta. Ela deve conectar o projeto do portfólio ao tipo de problema que aquele exemplo resolve.

Também vale adicionar tags. Com elas, você pode usar `Ctrl F` e encontrar rapidamente termos como `logo design`, `business card`, `editorial` ou `pixel art`.

Se o portfólio for grande, as tags economizam bastante tempo.

## Usando a central durante a proposta

Na prática, quando você encontra uma vaga de identidade visual, abre a central, busca por um projeto parecido e copia o link certo.

Depois, adapta a descrição para o cliente específico. A descrição não deve ser colada de forma automática, mas servir como base para mostrar por que aquele projeto é relevante para a vaga.

Esse cuidado evita erros como deixar placeholders, nomes incompletos ou textos genéricos demais.

O objetivo é ganhar velocidade sem perder personalização.

## Detalhes que ajudam na navegação

Mateus comenta que imagens pequenas ou ícones podem ajudar na visualização, especialmente se houver muitos projetos.

Eles não são obrigatórios, mas tornam o documento mais fácil de navegar.

Títulos bem organizados também criam uma barra lateral útil no documento. Assim, você pula rapidamente entre identidade visual, design gráfico, pixel art e outras áreas.

Quanto mais claro estiver o documento, menos energia você gasta procurando exemplos durante o processo de candidatura.

## Coloque em prática

Crie uma central de links com pelo menos três categorias de serviço.

Em cada categoria, adicione três projetos com nome, link, descrição curta e tags.

Depois, escolha uma vaga real e use a central para selecionar o exemplo de portfólio mais relevante para a proposta.
